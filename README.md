# Stock Market Predictor

A machine learning project to predict stock prices using historical data and various prediction models.

## Features

- Fetch historical stock data using yfinance
- Technical indicators (Moving Averages, Volatility, Daily Returns)
- Multiple ML models: LSTM, Random Forest, Gradient Boosting
- Data normalization and sequence creation for time series
- Model evaluation with metrics (MSE, MAE, RÂ²)
